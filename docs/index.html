Clash Premium 规则集 · 一键导入

选择适合你的模式，点击按钮即可导入 Clash 客户端

代理 黑名单 + 去广告

被墙网站走代理，其余直连，同时拦截广告。适合服务器质量一般、流量紧缺的用户。导入后需添加代理节点。

一键导入

手动导入：在 Clash 客户端的「订阅 / 配置下载」中粘贴上方链接

代理 白名单 + 去广告

常见直连网站直连，其余境外走代理，同时拦截广告。适合服务器稳定、流量充足的用户。导入后需添加代理节点。

一键导入

手动导入：在 Clash 客户端的「订阅 / 配置下载」中粘贴上方链接

去广告 仅去广告（推荐新手）

所有流量直连，仅拦截广告域名和 IP。无需代理节点，导入即用。适合不需要代理、只想去广告的用户。

一键导入

手动导入：在 Clash 客户端的「订阅 / 配置下载」中粘贴上方链接

使用说明
1. 确保已安装 Clash 客户端（如 Clash Verge Rev、FlClash、Clash Meta for Android 等）
2. 点击「一键导入」按钮，浏览器会自动唤起 Clash 客户端并导入配置
3. 如果一键导入没反应，点击「复制链接」，在 Clash 客户端中手动粘贴导入
4. 黑名单/白名单模式导入后，需在配置中添加自己的代理节点
5. 规则集每周一北京时间 06:00 自动更新，无需重新导入
6. 如果 raw.githubusercontent.com 无法访问，可将链接域名替换为 cdn.jsdelivr.net/gh
